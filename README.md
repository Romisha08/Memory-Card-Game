# Memory Card Game (React.js)

A simple and interactive Memory Card Matching Game built using React.js.  
Flip the cards, find matching pairs, and test your memory skills.

---

## Game Features

- Card flipping interaction  
- Random card shuffle on every game start  
- Match two identical cards to score points  
- Prevents clicking more than two cards at once  
- Tracks score and number of moves  
- Restart game option  
- Detects when the game is completed  

---

## Tech Stack

- React.js (Vite)
- JavaScript (ES6+)
- CSS
- React Hooks:
  - useState
  - useEffect
  - Custom Hook (useGameLogic)
